== Description ==

Sends user registrations,comments and posts to your Twitter account.

Most Twitter tools use CuRL PHP library. However, many hosting providers restrict the use of this library. Tweet Push use simple Twitter API's, no extra tools
or libraries needed.

Tweet Push can send new posts names, along with URL (optional). New comment summaries, with links to comments can also be send to Twitter. If you let users register to your blog, you can also send new user names and their profile URL's.

== Installation ==

Nothing fancy: just activate the plugin, and go to settings, fill your Twitter account credentials. You can enable/disable functions in the same screen.
